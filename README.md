# TeamTrack

Application where user can manage sport teams. 


# Requirements
Users can be player or capitan from each team.
Team has a type: Football, basketball, volleyball
For each team, has a number of members. (football 5, 7 or 11)
User has a limit of teams where can participate, this is base on the subscription.
Free subscription has a limit of 1 team, paid can be in 3 and pro in 5.
Be capitan or player count as 1 in order to be or manage teams for the subscription.
User can start a match and introduce stats during the match.
Stats has rules and points, for example, gol are 5 points, assists 3 points, yellow card -2 points, fault -1 point...
